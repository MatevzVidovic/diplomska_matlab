outMask = segmentation(filename);

As specified in:
http://www.ict.griffith.edu.au/conferences/btas2015/about-us.html

the format of the filename should be:
	E-xxx-y-z-n.jpg

In that case, the output mask will be written in a file named:
	M-xxx-y-z-n.jpg

located in the same directory of the input file.
